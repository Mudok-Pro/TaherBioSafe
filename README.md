# TaherBioSafe – Medical Waste Management Platform

## 🚀 Overview
TaherBioSafe is a web platform designed to manage and track medical waste processes in healthcare environments. It improves monitoring, organization, and traceability.

## ⚙️ Tech Stack
- React
- Next.js
- Firebase

## ✨ Features
- Medical waste tracking system
- Dashboard for monitoring operations
- Data management system
- Role-based structure (admin/user concept)
- Real-time database integration

## 🧠 Architecture
Data is stored in Firebase → users interact through dashboard → updates reflect in real time across the system.
