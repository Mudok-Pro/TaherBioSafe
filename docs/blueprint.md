# **App Name**: AshSync

## Core Features:

- Contract Request Form: Simple form for customers to request new contracts, including fields for all required information.
- Contract Management Dashboard: Customer dashboard where users can view and download their signed contracts.
- Pickup Ordering: Customers can order a pickup, selecting the waste type (chemical, biohazard, etc.).
- User Management: Admin dashboard for managing users, including adding, deleting, and changing roles.
- Report Summarization: AI-powered tool to summarize and categorize waste incineration reports for quick review by admins.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey professionalism and reliability.
- Secondary color: Light gray (#F0F0F0) for backgrounds and neutral elements.
- Accent: Emerald green (#50C878) for action buttons and key highlights.
- Clean and modern typography with good readability.
- Use a set of consistent and professional icons to represent different waste types, contract statuses, and actions.
- Clear and intuitive layouts for both customer and admin dashboards, with a focus on ease of navigation.
- Subtle transitions and animations to provide feedback on user interactions, such as form submissions and data updates.
